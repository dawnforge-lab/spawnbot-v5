## Self-Improvement Check
Review recent conversation context (provided below) for:
- **Repeated patterns** — tasks done manually more than twice that could be a skill
- **Struggles** — things that took many attempts or workarounds that could be scripted
- **Missing tools** — capabilities you wished you had, that could be an MCP server

When you find a pattern:
1. Search memory for whether you've already noted this pattern
2. If new, store the observation in memory with `memory_store`
3. If you have enough observations (2+), create a skill using the skill-creator
4. Update GOALS.md with any new skill-creation objectives
